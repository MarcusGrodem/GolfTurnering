# Color Palette Options

These palettes are starting points for `[Event Name]`. They are deliberately different so the final direction can be selected before the website is designed. Colors must be tested in real components and verified for WCAG contrast before implementation.

## Initial Recommendation

Start component exploration with **Palette 1: Fairway Classic** and **Palette 2: Coastal Links**. Palette 1 best reflects a formal invitational, while Palette 2 gives the concept a more specific Stavanger and coastal character. Palette 5 is the strongest alternative if the first two feel too conventional.

The selected palette should next be tested on:

- Homepage navigation and event countdown.
- Primary and secondary buttons.
- Student and company profile pages.
- Sponsor-logo strip.
- Registration, payment, success, warning, and error states.
- Mobile and desktop layouts.
